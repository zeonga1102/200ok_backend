# 녹턴앨리 B-street 지하2층 불법 입학 센터 (200ok)
해리포터 컨셉, 사람 사진을 초상화처럼 스타일을 변경하고 움직임을 추가해서 움직이는 초상화를 만들고 머신러닝을 통해 기숙사를 분류해주는 웹사이트

커뮤니티 기능 및 마이 페이지에서 방 꾸미는 기능 등
